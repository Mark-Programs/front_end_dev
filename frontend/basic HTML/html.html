Embedded Audio

Inline Elements:

  • controls: input this if you want controls shown
  • autoplay: input this if you want it to autoplay
  • preload: indicates what the browswer should do if the player is not set to autoplay
  • loop: input this to loop the audio
  • type: not commonly used
  • source: can specify more than one audio file as the source, within the audio tag. So if you want to reference an .mp3 and .ogg, you can and the browser can pick which one it wants (see code for second player).
    • MP3: Safari 5+, Chrome 6+, IE9
    • Ogg Vorbis: FF3.6, Chrome 6, Opera 1.5, IE9

Video

Video Elements

  • 'preload' values:
    • none: doesn't load video until play is pressed
    • auto: browser downloads the video when the pages loads
    • metadata: browser only gets data (size, first frame, track list, and duration
  • src: path to video
  • poster: cover image to show while it is downloading (jpg)
  • width, heigh: sets player size
  • controls: add this element to show controls
  • autoplay: plays video once it loads
  • loop: loops the video
NOTE: can also select multiple sources, same as audio, can also specify codecs required.
