Responding to Users
:hover, :active, :focus
Hover is applied when a user hovers over the element,
Active is applied when it gets activated by the user,
Focus is applied when an element has focus(when a browser discovers you are ready to interact with an element: IE when a cursor is in a form input ready to accept typing)
